## ✨ Features

### 👤 User Features

* Register & Login (JWT Authentication)
* Browse Movies & Series
* Search & Filter (Genre, Rating, Platform)
* Add to Watchlist
* Submit Reviews (1–10 rating)
* Like & Comment on Reviews
* Purchase / Subscription System

### 👑 Admin Features

* Add / Edit / Delete Movies
* Approve or Unpublish Reviews
* Manage Users & Content
* View Analytics
